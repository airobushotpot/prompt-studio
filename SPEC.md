# Prompt Studio — 产品规格说明书

## 1. 产品概述

**产品名称**: Prompt Studio
**类型**: 跨平台桌面/移动应用
**一句话描述**: 可视化管理和编辑 AI 提示词的工具，帮助用户创建、组织、测试和分享提示词。
**首要平台**: macOS

## 2. 技术栈

- **框架**: Tauri 2.x（Rust 后端 + Web 前端）
- **前端**: React 18 + TypeScript + Vite
- **样式**: TailwindCSS + shadcn/ui
- **状态管理**: Zustand
- **数据库**: SQLite（本地），通过 Tauri SQL 插件
- **提示词编辑**: TipTap（富文本编辑器，支持变量高亮）
- **ID 生成**: nanoid
- **图标**: Lucide React

## 3. 核心功能

### 3.1 提示词管理（CRUD）
- 创建新提示词（标题 + 内容 + 描述）
- 编辑提示词（富文本编辑器）
- 删除提示词（软删除，进入回收站）
- 复制提示词（克隆）

### 3.2 组织系统
- **文件夹**: 创建/重命名/删除文件夹，支持嵌套
- **标签**: 给提示词打多个标签
- **收藏**: 收藏常用提示词
- **回收站**: 软删除的提示词可恢复或彻底删除

### 3.3 提示词编辑器
- 富文本编辑（TipTap）
- **变量系统**: `{{variable_name}}` 语法，自动高亮显示
- 变量面板：填入每个变量的值，生成最终提示词
- 实时预览：看 AI 会收到什么
- 历史版本：每次保存生成快照，可回滚

### 3.4 模板系统
- 预设模板库（通用、AI 写作、代码生成、角色扮演等分类）
- 用户可保存自己的模板
- 从模板创建新提示词

### 3.5 搜索与过滤
- 全文搜索（标题 + 内容）
- 按文件夹过滤
- 按标签过滤
- 按收藏/最近使用过滤

### 3.6 导入/导出
- 导出为 JSON（完整备份）
- 导出为 Markdown
- 从 JSON 导入
- 从 Markdown 导入

### 3.7 设置
- 主题：浅色 / 深色 / 跟随系统
- 快捷键自定义
- 数据存储路径
- 清除本地数据

## 4. 数据模型

### Prompt
```
id: string (nanoid)
title: string
content: string
description: string
folder_id: string | null
tags: string[]
is_favorite: boolean
is_deleted: boolean
created_at: timestamp
updated_at: timestamp
```

### Folder
```
id: string (nanoid)
name: string
parent_id: string | null
created_at: timestamp
```

### Tag
```
id: string (nanoid)
name: string
color: string (hex)
```

### PromptVersion (历史快照)
```
id: string (nanoid)
prompt_id: string
content: string
created_at: timestamp
```

### Variable
```
id: string (nanoid)
name: string
default_value: string | null
```

## 6. 平台优先级

1. **macOS** (v1.0) — 首要目标，使用 Tauri + NSWindow
2. **Windows** (v1.1) — Tauri 支持 Windows
3. **Web** (v1.2) — 把 Tauri 的 webview 打包成 PWA
4. **iOS/Android** (v2.0) — 复用 web 版本，打包成 App

## 8. MVP 范围（第一版）

第一版只做核心功能，不做模板市场和云同步：

✅ 创建/编辑/删除提示词
✅ 文件夹管理
✅ 标签系统
✅ 收藏
✅ 变量系统（{{var}} 语法）
✅ 变量填充面板
✅ 搜索和过滤
✅ 深色/浅色主题
✅ 导入/导出 JSON

❌ 历史版本（后续版本）
❌ 模板市场（后续版本）
❌ 云同步（后续版本）
❌ 移动端（后续版本）

## 9. 验收标准

- macOS 上能正常打开窗口
- 能创建、编辑、删除提示词
- 变量 {{xxx}} 能被正确识别和高亮
- 变量填充后能生成最终文本
- 数据保存在本地 SQLite
- 深色/浅色主题能切换
- 能导出和导入 JSON
